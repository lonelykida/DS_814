# 文档说明

## 一、文档包含内容说明

```txt
1.readme.md -->说明文档
2.test.hpp  -->代码主体文件 - 数据结构代码
3.test.cpp  -->测试文件 - main
4.数据结构知识点_1.md  -->数据结构所有知识点
5.run.exe   -->test.cpp生成的二进制文件
.vscode     -->vscode配置文件
```



## 二、使用说明

1.下载test.hpp到本地，添加到VS或VScode打卡即可。

2.test.cpp是包含测试代码的文档，但测试代码在编写时未保存，因此可能只有最近测试的代码在上边，若有需要也可以下载使用。
